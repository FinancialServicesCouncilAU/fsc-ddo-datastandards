Element ComplaintsReport

This element defines the root element of the report

Element ReportIdentifier

Report ID – unique to distributor. Can be combined with distributor ID to have industry-wide unique ID

Element ReportExtractionDateTime

Date that transaction report was made. If report is reissued, the new information replaces any reports within the defined report start and end period.

Element ReportSelectionRange

Defines the range the complaints report covers

Attribute StandardVersion

Should automatically pick up the Standard's current version

Element ComplaintEntities

List of all entities who are impacted by the Complaint report

Element EntityRelationships

List of key relationships between entities in order to build the entity hierarchy

Element ReportingEntityIdentifier

Identifier from the listed Complaint Entities that is filing this report

Attribute NilReportCount

Number of nil complaint records included in this file. Sum of this field and the NonNilReportCount attribute must equal number of complaint records in this file.

Attribute NonNilReportCount

Number of non-nil complaint records included in this file.

Element ComplaintId

Reporting entity internal code for complaint

Element IssuerIdentifer

A primary identifier from one of the ComplaintEntities

Element DistributorIdentifier

A Primary identifer from an entity in the ComplaintEntities

Element HasFinancialImpact

Element ComplaintDate

Date complaint was raised

Element ComplaintResolutionDate

Leave blank (ie use xs:Nil) if not resolved or unknown

Element ComplaintContent

This should include all the relevant details so that the issuer can understand the nature of the complaint
